# Lab Maya Informatika – Representasi Gambar

Paket ini memakai struktur dan gaya antarmuka dari template `final-v2-python-data-lab`, lalu diadaptasi penuh menjadi materi **Representasi Gambar**.

## Isi utama
- Beranda dengan menu: Tujuan Pembelajaran, Cara Penggunaan, Mulai Percobaan, Biodata Pengembang
- Tombol Mulai terkunci sampai Tujuan dan Cara Penggunaan dibaca
- Pop-up konsep dasar muncul saat pertama kali masuk lab
- Lab piksel 10×10 dengan 5 tahap bertahap
- Modal Contoh Tahap, Istilah Penting, Refleksi, Biodata Pengembang, dan Kuis Refleksi
- Aman untuk GitHub Pages dan penggunaan offline

## Struktur
- `index.html`
- `assets/src/style.css`
- `assets/src/script.js`
- `assets/img/*`

## Catatan
- Biodata masih mengikuti contoh / placeholder dan bisa diganti kapan saja
- Jika ingin mengganti identitas final, cukup ubah isi modal biodata pada `index.html`
