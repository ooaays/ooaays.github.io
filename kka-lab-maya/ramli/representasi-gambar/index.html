Representasi Gambar

Informasi Materi

Tujuan Praktikum

Untuk memahami bagaimana data dalam situasi kehidupan sehari-hari dapat dikelola, disederhanakan, dan direpresentasikan menggunakan sistem biner (0 dan 1) sehingga mudah disimpan, diproses, dan ditampilkan.

Manipulator Variabel

  • 0 = tidak ada / tidak aktif / kosong
  • 1 = ada / aktif / terisi
  • Tombol arah = memindahkan kursor
  • Hapus = mengosongkan pixel

Hasil

Data biner pada grid 10 x 10 dapat menampilkan pola atau gambar.

Target Gambar

Mode bebas aktif. Sekarang tidak ada target, jadi silakan membuat gambar versimu sendiri.

Image Matrix 10 x 10

Klik kotak atau gunakan tombol arah lalu tekan 0 / 1

Binary Data Stream

Petunjuk Singkat

  • Pilih posisi pixel pada grid
  • Tekan 1 untuk mengisi pixel
  • Tekan 0 untuk mengosongkan pixel
  • Perhatikan perubahan data biner
  • Susun hingga menjadi gambar

Berpikir Komputasional

Capaian Pembelajaran

Menerapkan pengelolaan data, pemecahan masalah sederhana dalam kehidupan masyarakat secara sistematis, dan menuliskan instruksi.

Tujuan Pembelajaran

Untuk memahami bagaimana data dalam situasi kehidupan sehari-hari dapat dikelola, disederhanakan, dan direpresentasikan menggunakan sistem biner (0 dan 1) sehingga mudah disimpan, diproses, dan ditampilkan.

Cara Penggunaan

1. Mengenal Tampilan Lab Maya

Lab maya terdiri dari tiga bagian utama:

1️⃣ Panel Informasi

Berisi judul materi, tujuan praktikum, manipulator variabel, dan hasil yang diharapkan.

2️⃣ Image Matrix

Grid 10 x 10 untuk menyusun pixel menjadi pola atau gambar.

3️⃣ Binary Data Stream

Menampilkan data 0 dan 1 sesuai kondisi pixel.

2. Fungsi Tombol Kontrol

Tombol Fungsi
0Kondisi tidak ada / tidak aktif / kosong
1Kondisi ada / aktif / terisi
Atas, Bawah, Kiri, KananMemindahkan posisi kursor pada grid
HapusMenghapus pixel yang dipilih

3. Langkah Percobaan

  1. Arahkan kursor ke salah satu kotak pada grid.
  2. Tekan tombol 1 untuk mengaktifkan pixel.
  3. Tekan tombol 0 untuk mengosongkan pixel.
  4. Amati perubahan pada grid dan data biner.
  5. Susun beberapa pixel aktif hingga membentuk gambar sederhana.

4. Hasil yang Diharapkan

Data biner yang disusun dalam suatu sistem, misalnya grid pixel 10 x 10, dapat menampilkan informasi visual seperti pola atau gambar.

💡 Susunan 0 dan 1 bisa membentuk tampilan visual. Komputer memang bekerja dari detail kecil yang terus diulang sampai manusia menyebutnya gambar.

5. Tips Praktikum

  • ✔ Perhatikan posisi baris dan kolom saat mengisi pixel.
  • ✔ Bandingkan selalu Image Matrix dengan Binary Data Stream.
  • ✔ Gunakan tombol Hapus jika ada pixel yang salah.
  • ✔ Susun data biner dengan teliti agar gambar yang terbentuk sesuai harapan.
