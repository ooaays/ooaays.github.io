# Python Data Lab v2 — Detektif Data Kantin

Topik: Teknologi untuk Mengolah Data — Berpikir Komputasional dengan Python.

## Cara Menjalankan

Buka file index.html langsung di browser. Tidak perlu internet.

## Tahapan Belajar (5 Tahap)

Tahap 1: Rapikan Data — list, variabel
Tahap 2: Cari Terbesar — max()
Tahap 3: Saring Data — if, kondisi
Tahap 4: Tampilkan Semua — for, perulangan
Tahap 5: Gabungkan — zip() (BARU)

## Fitur 

- Tahap 5: Fungsi zip()
- Progress Bar visual
- Kuis Refleksi: 5 soal kontekstual
- Sertifikat: bisa dicetak
- Mode Guru: Ctrl+Shift+G atau klik logo 5x
- Kotak Konsep Kunci per tahap
- Offline 100%
