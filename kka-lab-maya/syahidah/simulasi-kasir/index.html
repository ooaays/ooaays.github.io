Tujuan Praktikum

Sebelum melakukan praktikum, murid sudah mendapatkan materi tentang:

  1. Variabel dan Tipe Data Numerik
  2. Operator Aritmatika Dasar
  3. Perintah Input dan Output (I/O)
Praktikum bertujuan agar murid dapat menuliskan instruksi menggunakan teknologi atau pemrograman dasar untuk memodelkan proses transaksi kasir sederhana, termasuk mengatur variabel, menghitung total harga, dan menentukan kembalian secara interaktif..

Cara Penggunaan

1. Mengenal Tampilan Lab Kasir

Lab virtual ini menampilkan simulasi sebuah toko dengan kasir dan pembeli.

  • Pembeli membawa keranjang belanja
  • Troli berisi barang yang akan dihitung
  • Kasir yang akan memproses pembayaran
  • Balon percakapan untuk menampilkan hasil program

Simulasi ini digunakan untuk memahami bagaimana sebuah program kasir bekerja.

2. Tujuan Percobaan

Dalam lab ini kamu akan membuat program kasir sederhana menggunakan konsep Python.

  • Menggunakan variabel untuk menyimpan data harga
  • Menggunakan operator matematika untuk menghitung total
  • Menghitung kembalian uang pembeli
  • Menampilkan hasil transaksi

3. Variabel yang Digunakan

Program kasir menggunakan beberapa variabel berikut:

  • harga_roti → harga satu roti
  • harga_susu → harga satu susu
  • jumlah_roti → jumlah roti yang dibeli
  • jumlah_susu → jumlah susu yang dibeli
  • uang_pembeli → uang yang dibayar pembeli

4. Menghitung Total Belanja

Total belanja dihitung menggunakan operator matematika.

total = (jumlah_roti × harga_roti) + (jumlah_susu × harga_susu)

Program akan menjumlahkan seluruh harga barang yang dibeli.

5. Menghitung Kembalian

Setelah total belanja diketahui, program akan menghitung kembalian.

kembalian = uang_pembeli - total

Jika uang pembeli lebih besar dari total belanja, maka kasir memberikan kembalian.

6. Fungsi Sapaan Kasir

Program juga memiliki fungsi untuk menyapa pembeli.

def menyapa(): return "Halo, selamat datang!"

Fungsi ini akan menampilkan pesan dari kasir kepada pembeli.

7. Menjalankan Program

Setelah semua variabel dan rumus dimasukkan, program akan dijalankan.

  • Kasir menyapa pembeli
  • Program menghitung total belanja
  • Program menghitung kembalian
  • Hasil transaksi ditampilkan

8. Hasil Program

Jika program berjalan dengan benar, maka akan muncul hasil seperti berikut:

Halo, selamat datang! Total Belanja: Rp 17000 Kembalian: Rp 3000 Terima kasih telah berbelanja!

9. Tips Praktikum

  • Perhatikan nilai variabel yang digunakan.
  • Gunakan operator matematika dengan benar.
  • Pastikan perhitungan total dan kembalian sesuai.
  • Coba ubah jumlah barang untuk melihat hasil berbeda.
